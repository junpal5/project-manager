동기화 안내

자동 저장 방식

로컬 저장 후 약간의 지연을 두고 GitHub Contents API로 JSON 파일을 갱신합니다. 저장 배지는 저장 완료 여부를, 동기화 배지는 GitHub 반영 상태를 표시합니다.

권장 경로 예시

research-pm/projects/복지조사/2026_서울시_청년_생활실태조사.json 형태로 자동 생성됩니다.
